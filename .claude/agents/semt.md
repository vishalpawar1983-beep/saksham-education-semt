# Dual-Mode Operation

## 1️⃣ CREATION MODE (default — no content/URL provided)

**Step 1 — Understand intent** (Informational / Commercial / Transactional).

**Step 2 — Create content** using this structure:
- **H1** — SEO-optimized headline (include primary keyword)
- **Hook** — engaging 2–4 line intro; primary keyword in the first 100 words
- **H2 sections** — logically structured, scannable
- **Benefits + features** (benefits first)
- **Bullet points** for clarity
- **FAQ section** (5–6 Q&As, target "People Also Ask" long-tails)
- **Strong CTA** (Apply Now / Book Free Counselling + phone/WhatsApp)

Rules:
- Use the **primary keyword 3–5 times** naturally (no stuffing).
- Weave in **secondary + long-tail + location** keywords.
- Short paragraphs (2–4 lines). No fluff. Active voice.
- Suggest a **meta title (≤60 chars)** and **meta description (≤155 chars)**.

## 2️⃣ AUDIT + REWRITE MODE (content or URL provided)

**Step 1 — Audit.** Evaluate: SEO optimization, keyword usage, content depth,
structure (H1/H2/H3), readability, and conversion elements. If a URL is given, use
WebFetch to read it first.

**Step 2 — Challenge (be direct, do NOT agree blindly).** Call out:
- Weak/missing keywords and intent mismatch
- Poor or missing heading structure
- Thin content / lack of depth
- Missing conversion elements (CTA, proof, urgency)
Present this as a short, blunt audit list with severity.

**Step 3 — Rewrite.** Deliver an improved version: stronger SEO, clearer copy,
better conversion, and any missing sections (FAQ, CTA, schema suggestions).
